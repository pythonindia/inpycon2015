Jiva™ is a web architected service with a web frontend, application server farm and RDBMS for transaction storage, is designed in Model View Controller (MVC) architecture using Python web framework, which supports thin-client environment. Jiva™ components can operate individually and can be invoked via web services to create service oriented architecture (SOA). Jiva™ supports Oracle and Microsoft SQL Server RDBMS and its application servers can be hosted on Windows or Linux servers. Jiva™ can be configured for continuous availability and allows for infinite scalability horizontally or vertically. Jiva™ is delivered on virtual Intel servers running Windows or Linux with simple pre-installation set up and described in specific documents for each architecture. Jiva™ is tested and certified on all modern browsers. Jiva™ utilizes a number of technologies, such as AngularJS, MongoDB, Memcache, SQLAlchemy ORM, RabbitMQ that adhere to industry standards.
